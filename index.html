Analytics

OpenMF analytics is a mobile forensic analytics service offered by SCoRe Lab (Sustainable Computing Research Lab) that helps to analyze data extracted from Android Devices.

Common Words: Search common between the data.
Keyword Search: Search keywords to relate the cases from the overall database as well as from particular case.
Filter: Filter cases date-wise or by tags.
Location: Find the searched location from the case.
Data Visualization: Visualize data using maps, graphs, and tables from the case.
Relate one case with another using graphs, maps, and tables. Common locations on map, common browser history with frequency on Graph, common call logs, text messages using tables are available on OpenMF front end.

Data Extraction

OpenMF is currently able to extract data of SMS(text messages), Facebook, WhatsApp, Browser history, Bluetooth data, EXIF data from images, Location history, General information of the device, Call logs, and Contacts. Data can be extracted from CMD as well as using API(POSTMAN) or from Frontend also.

  • SMS: Data like thread_id, address, date, date_sent, read, body, creator, seen, sim_slot, sim_imsi.
  • Facebook: Data like text, contacts, senders name, timestamp, profile id.
  • WhatsApp: Data like mobile number, time, text, type, media name.
  • Browser History: Data like URLs, URLs title, Visit count, Urls last visit time, hidden, visit time, visit duration, visits transition, visit source.
  • Bluetooth: Data like shared file path, name of the file, data, type, direction, destination, visibility, status, total bytes, timestamp, device name.
  • EXIF data: Data from images like file path, size, date taken, mime_type, title, description, display name, Picasa id, orientation, latitude, longitude, date added, date modified, mini_thumb_magic, bucket id, bucket display name, album, tags, category, language, storage id, width, and height.
  • Location: Data like latitude, longitude, timestamp, and saved location link.
  • General Information: Data like Shell permission, ADB serial, Manufacturer, Model number, IMEI, Android Version, Build name, Wifi MAC, Local time, Android time, Accounts, Case Name, and Tags.
  • Call logs: Data like number,name,date,duration,type,location, and frequent.
  • Contacts: Data like phone number and display name.
